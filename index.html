How to Use This Demo

  • Step 1: Load Data - Fetches the MovieLens 100K dataset (interactions, movie titles, genres, and user info). This may take a few seconds.
  • Step 2: Train Both Models - Trains both a simple embedding model and a deep model. This is computationally intensive and can take several minutes, depending on your computer's performance. The browser may become unresponsive during training.
  • Step 3: Test & Compare - After training, this button picks a random user and shows three lists: their actual highest-rated movies, recommendations from the simple model, and recommendations from the deep model.

How to Host on GitHub Pages

  1. Create a new public GitHub repository.
  2. Upload index.html, app.js, and two-tower.js to the root of the repository.
  3. Create a data folder in the repository.
  4. Download the MovieLens 100K dataset and place u.data, u.item, and u.user inside the data folder.
  5. In your repository settings, go to the "Pages" section and enable GitHub Pages for your main branch.
